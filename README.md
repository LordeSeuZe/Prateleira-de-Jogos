# Prateleira-de-Jogos
Parte 1, atividade de CPW4, 2026/1

# Tema: Prateleira de jogos
O projeto deve funcionar como uma expecie de filho perdido do aplicativo Letterboxd, mas com a difereça que ao em de vez de serem postos os seus filmes favoritos, serão adicionados jogos.

A Api uzada foi RAWG, onde pode utilizar uma ampla gama de jogos, oque auxiliou em contra tempos com os planos iniciais de usar uma api propria.
Como já tinha visto essa api sendo em projetos de amigos, preferi utilizada-la
